# 🏦 金融數據分析網頁

本專案為一套基於 Django 開發的金融數據分析網頁應用，整合股票資料蒐集、資料分析與視覺化功能。系統透過 yfinance、Selenium 等工具取得金融資料，並使用 HighCharts 與 DataTables 進行圖表呈現，提供使用者多種股票分析功能與互動式操作介面。

---

# 📍 功能介紹

## 1. 爆大量分析
透過取得股票歷史成交量資料並計算五日移動平均線（MA），判斷成交量是否出現「爆大量」現象，協助使用者觀察市場熱度與流動性變化。分析結果以蠟燭圖與表格方式呈現。

---

## 2. 股票定價策略
提供多種股票估值方法，包括：

- 股利法  
- 高低價法  
- 本淨比法（PBR）  
- 本益比法（PER）

系統會根據歷史資料計算便宜價、合理價與昂貴價，協助使用者評估股票投資價值。

---

## 3. 本益比河流圖
利用 **PER River Chart（本益比河流圖）** 呈現股票在不同本益比區間下的價格範圍，幫助使用者判斷目前股價所處的估值區間。

---

## 4. 天花板線與地板線分析
透過不同計算方法推估股票價格的 **上限（天花板）與下限（地板）**，協助分析股價可能的極端範圍與支撐壓力位置。

---

## 5. 技術指標進出場分析
整合多種常見技術分析指標，例如：

- KD 指標
- MACD 指標
- 布林通道（Bollinger Bands）

系統會透過圖表標示潛在買賣訊號，協助使用者判斷市場趨勢。

---

## 6. 帳號登入與驗證功能
實作使用者帳號系統，包含：

- 使用者註冊
- 使用者登入
- 未登入使用者強制導向登入頁面

透過資料庫管理使用者資訊並保護系統功能。

---

## 7. Tracker 與 Email 通知功能
提供使用者建立個人化追蹤清單（Tracker），使用者可以追蹤特定股票分析結果。當系統偵測到技術指標產生訊號時，會自動寄送 Email 通知給使用者。

---

## 8. Docker 容器化部署
使用 Docker 將整個系統進行容器化封裝，包含 Web 服務與資料庫環境，使系統能在不同環境中快速部署與執行。
